iTunes Scrobbling

Download the script

This is a script developed by a fellow Universal Scrobbler user, drailskid. It is an iTunes Applescript plugin which automatically scrobbles any selected tracks in iTunes to your account when it is run. It will work as long as there is a keychain entry in the default keychain with the name "last.fm" with your username and password. If you're not familiar with AppleScripts in iTunes, you can read about them here.

This is a Mac-only script, and as a Windows user I have not been able to use it, so I cannot guarantee it works. If you have any questions, please direct them to the author at drailskid@yahoo.com.

Props to drailskid for this script and for sharing his hard work! Similarly, if you have any US-related scripts/online tools you'd be happy to share, please pass them on to me.
